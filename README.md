### Demo Project: [C]ontinuous [I]ntegration with Jenkins

#### Objectives
Using Jenkin's 
- Multibranching 
- Jenkins Shared Library usage written in Groovy for clean 
- Version updating after a success test on dev branch
- Merging dev branch to main
- OCI creation and uploading to Docker Hub private repository
- Webhook to auto CI pipeline triggering

#### Used Technologies :
Jenkins, Groovy, Podman, Git, Java, Maven

Jenkins will run as a container on Podman runtime over Rocky 10.
